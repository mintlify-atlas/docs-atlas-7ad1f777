QuestPDF is distributed as a NuGet package and supports all major .NET runtimes and platforms.

## Requirements

| | Supported |
|---|---|
| **Frameworks** | `net6.0`, `net8.0`, `net10.0`, `netstandard2.0` (.NET Framework 4.6.2+) |
| **Operating systems** | Windows, Linux, macOS |
| **Cloud & containers** | Azure, AWS, Google Cloud, Docker, Kubernetes |

## Install the package

<CodeGroup>

```bash dotnet CLI
dotnet add package QuestPDF
```

```powershell Package Manager Console
Install-Package QuestPDF
```

```xml .csproj ItemGroup
<ItemGroup>
  <PackageReference Include="QuestPDF" Version="2026.2.4" />
</ItemGroup>
```

</CodeGroup>

<Note>
  The current release is **2026.2.4**. Check [NuGet](https://www.nuget.org/packages/QuestPDF) for the latest version.
</Note>
